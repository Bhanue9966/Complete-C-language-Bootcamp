# printf()

## 1. Introduction

In programming, a program often needs to communicate with the user by displaying messages, results, or instructions on the screen.

For example:

- Display a welcome message.
- Show the result of a calculation.
- Print the value of a variable.
- Display error messages.
- Format reports and tables.

In C, the most commonly used function for displaying output is **`printf()`**.

The name **`printf`** stands for **Print Formatted**.

It belongs to the Standard Input/Output Library (`stdio.h`).

---

# 2. Definition

`printf()` is a standard library function in C used to display formatted output on the console (screen).

It can print:

- Text
- Numbers
- Characters
- Floating-point values
- Variable values
- Expressions
- Formatted tables

---

# 3. Why Do We Need `printf()`?

Imagine an ATM machine.

If the ATM never displayed messages such as:

```
Insert Card
Enter PIN
Transaction Successful
```

the user would never know what was happening.

Similarly, a C program uses `printf()` to communicate with the user.

Without `printf()`, the program may execute correctly, but the user cannot see its results.

---

# 4. Real-Life Analogy

Think of a teacher in a classroom.

The teacher knows the answers but must **speak** so that students can understand.

```
Computer
     │
     ▼
 printf()
     │
     ▼
 Screen
     │
     ▼
 User
```

`printf()` acts as the "voice" of the program.

---

# 5. Header File Required

Before using `printf()`, include:

```c
#include <stdio.h>
```

Why?

Because `printf()` is declared in the `stdio.h` header file.

---

# 6. Syntax

```c
printf("text");
```

General syntax:

```c
printf("format string", arguments);
```

Examples:

```c
printf("Hello");
```

```c
printf("%d", age);
```

```c
printf("%f", salary);
```

---

# 8. Example 1 – Printing Text

```c
#include <stdio.h>

int main()
{
    printf("Hello, World!");

    return 0;
}
```

### Output

```
Hello, World!
```

### Explanation

- `printf()` displays the text exactly as written.
- The text must be enclosed in double quotes (`"`).

### Conclusion

`printf()` can print plain text directly to the screen.

---

# 9. Example 2 – Printing Multiple Lines

```c
#include <stdio.h>

int main()
{
    printf("Welcome\n");
    printf("to\n");
    printf("C Programming");

    return 0;
}
```

### Output

```
Welcome
to
C Programming
```

### Explanation

`\n` is the newline escape sequence. It moves the cursor to the next line.

### Conclusion

Use `\n` to improve the readability of output.

---

# 10. Example 3 – Printing Variables

```c
#include <stdio.h>

int main()
{
    int age = 21;

    printf("Age = %d", age);

    return 0;
}
```

### Output

```
Age = 21
```

### Explanation

- `%d` is a format specifier for integers.
- `age` replaces `%d` in the output.

### Conclusion

Use format specifiers to display variable values.

---

# 11. Example 4 – Printing Multiple Variables

```c
#include <stdio.h>

int main()
{
    char grade = 'A';
    int marks = 95;

    printf("Grade = %c\n", grade);
    printf("Marks = %d", marks);

    return 0;
}
```

### Output

```
Grade = A
Marks = 95
```

### Conclusion

Different data types require different format specifiers.

---

# 12. Example 5 – Printing Floating-Point Values

```c
#include <stdio.h>

int main()
{
    float salary = 45678.75;

    printf("Salary = %f", salary);

    return 0;
}
```

### Output

```
Salary = 45678.750000
```

### Explanation

By default, `%f` prints six digits after the decimal point.

---

# 13. Example 6 – Controlling Decimal Places

```c
#include <stdio.h>

int main()
{
    float pi = 3.14159265;

    printf("%.2f", pi);

    return 0;
}
```

### Output

```
3.14
```

### Explanation

`%.2f` tells `printf()` to display two digits after the decimal point.

### More Examples

```c
%.1f
```

```
3.1
```

```c
%.3f
```

```
3.142
```

```c
%.5f
```

```
3.14159
```

---

# 14. Example 7 – Printing Expressions

```c
#include <stdio.h>

int main()
{
    int a = 10;
    int b = 20;

    printf("Sum = %d", a + b);

    return 0;
}
```

### Output

```
Sum = 30
```

### Explanation

`printf()` can display the result of an expression directly without storing it in another variable.

# 15. Common Format Specifiers

| Format Specifier | Meaning | Example Value |
|------------------|---------|---------------|
| `%d` | Integer | 25 |
| `%f` | Float | 25.75 |
| `%lf` | Double | 125.456 |
| `%c` | Character | A |
| `%s` | String | Hello |
| `%u` | Unsigned Integer | 100 |
| `%x` | Hexadecimal | 1A |
| `%o` | Octal | 17 |
| `%%` | Percent Symbol | % |

> We'll explore format specifiers in much greater detail in the dedicated **Input and Output** module.

# 16. Common Escape Sequences

| Escape Sequence | Meaning |
|-----------------|---------|
| `\n` | New line |
| `\t` | Horizontal tab |
| `\\` | Backslash |
| `\"` | Double quote |
| `\'` | Single quote |
| `\b` | Backspace |
| `\r` | Carriage return |

Example:

```c
printf("Name\tAge\n");
printf("John\t25");
```

Output:

```
Name    Age
John    25
```

# 17. Behind the Scenes

When you write:

```c
printf("Hello");
```

the following happens conceptually:

```
Your Program
      │
      ▼
Calls printf()
      │
      ▼
C Standard Library
      │
      ▼
Operating System
      │
      ▼
Console (Terminal)
      │
      ▼
Screen Displays:
Hello
```

`printf()` does not print directly to the monitor. It sends the output through the C Standard Library and operating system to the terminal.

# 18. Common Beginner Mistakes

### Missing Header File

```c
printf("Hello");
```

without:

```c
#include <stdio.h>
```

may result in compiler warnings or errors.

---

### Forgetting Double Quotes

❌ Incorrect

```c
printf(Hello);
```

✔ Correct

```c
printf("Hello");
```

---

### Wrong Format Specifier

```c
int age = 20;

printf("%f", age);
```

This produces incorrect output because `%f` expects a floating-point value.

---

### Forgetting the Semicolon

```c
printf("Hello")
```

Compiler Error:

```
expected ';'
```

---

# 19. Best Practices

- Always include `<stdio.h>` when using `printf()`.
- Use the correct format specifier for each data type.
- Add `\n` where appropriate to keep the output readable.
- Write meaningful output messages.
- Format output neatly for better user experience.
