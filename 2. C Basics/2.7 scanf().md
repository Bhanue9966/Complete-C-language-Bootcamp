# scanf()

## 1. Introduction

Most real-world programs need input from the user.

For example:

- Enter your name.
- Enter your age.
- Enter two numbers.
- Enter your salary.
- Enter your exam marks.

Instead of using fixed values inside the program, we can ask the user to provide input while the program is running.

In C, the most commonly used function for accepting keyboard input is **`scanf()`**.

The name **`scanf`** stands for **Scan Formatted**.

It belongs to the Standard Input/Output Library (`stdio.h`).

---

# 2. Definition

`scanf()` is a standard library function used to read formatted input from the keyboard and store it in variables.

It allows the user to enter values while the program is running.

---

# 3. Why Do We Need `scanf()`?

Imagine filling out an online registration form.

The website asks you to enter:

- Name
- Age
- Email
- Phone Number

The website cannot guess these values.

Similarly, a C program cannot know the user's input unless it asks for it.

`scanf()` enables the program to receive information from the user.

---

# 4. Real-Life Analogy

Think of a classroom.

```
Teacher

↓

Asks a Question

↓

Student Answers

↓

Teacher Records the Answer
```

Similarly,

```
Program

↓

scanf()

↓

User Types Input

↓

Value Stored in Variable
```

`scanf()` acts as the listener that records the user's input.

---

# 5. Header File Required

Before using `scanf()`, include:

```c
#include <stdio.h>
```

---

# 6. Basic Syntax

```c
scanf("format_specifier", &variable);
```

Example:

```c
scanf("%d", &age);
```

Here:

- `%d` → Format specifier for an integer.
- `&age` → Address of the variable where the input will be stored.

---

# 7. Understanding Each Part

Example:

```c
scanf("%d", &age);
```

### `scanf`

The function used to read formatted input.

---

### `%d`

Tells `scanf()` to expect an integer.

---

### `&`

The **address-of operator**.

It tells the function where the variable is located in memory.

---

### `age`

The variable that will store the input.

---

# 8. Why Do We Use `&`?

Suppose you want a friend to deliver a package to your house.

Simply saying:

```
Bhanu
```

is not enough.

You need to provide the **address** of your house.

Similarly,

`scanf()` needs the **memory address** of the variable so it knows where to store the value entered by the user.

Example:

```c
scanf("%d", &age);
```

Here,

`&age` means:

> "Store the entered value in the memory location of `age`."

> **Important:** You will learn the address-of operator and pointers in detail in the Pointers module. For now, remember that `&` is generally required when reading values into variables with `scanf()`.

---

# 9. Example 1 – Reading an Integer

```c
#include <stdio.h>

int main()
{
    int age;

    printf("Enter your age: ");

    scanf("%d", &age);

    printf("Your age is %d", age);

    return 0;
}
```

### Sample Output

```
Enter your age: 21
Your age is 21
```

### Explanation

- The program asks the user for an age.
- `scanf()` reads the integer.
- The value is stored in `age`.
- `printf()` displays the stored value.

### Conclusion

`scanf()` allows programs to receive user input.

---

# 10. Example 2 – Reading a Floating-Point Number

```c
#include <stdio.h>

int main()
{
    float salary;

    printf("Enter your salary: ");

    scanf("%f", &salary);

    printf("Salary = %.2f", salary);

    return 0;
}
```

### Sample Output

```
Enter your salary: 45678.75
Salary = 45678.75
```

### Conclusion

Use `%f` to read `float` values.

---

# 11. Example 3 – Reading a Character

```c
#include <stdio.h>

int main()
{
    char grade;

    printf("Enter your grade: ");

    scanf(" %c", &grade);

    printf("Grade = %c", grade);

    return 0;
}
```

### Sample Output

```
Enter your grade: A
Grade = A
```

### Explanation

Notice the space before `%c`:

```c
" %c"
```

It tells `scanf()` to skip leading whitespace (such as spaces or newline characters) before reading the character.

### Conclusion

Using a leading space before `%c` helps avoid common input issues.

---

# 12. Example 4 – Reading Multiple Values

```c
#include <stdio.h>

int main()
{
    int a, b;

    printf("Enter two numbers: ");

    scanf("%d %d", &a, &b);

    printf("First Number = %d\n", a);
    printf("Second Number = %d", b);

    return 0;
}
```

### Sample Output

```
Enter two numbers: 10 20
First Number = 10
Second Number = 20
```

### Explanation

`scanf()` can read multiple values in a single call.

---

# 13. Example 5 – Simple Calculator

```c
#include <stdio.h>

int main()
{
    int num1, num2;

    printf("Enter two numbers: ");

    scanf("%d %d", &num1, &num2);

    printf("Sum = %d", num1 + num2);

    return 0;
}
```

### Sample Output

```
Enter two numbers: 15 25
Sum = 40
```

### Conclusion

Programs become interactive by accepting user input.

# 14. Common Format Specifiers

| Format Specifier | Data Type |
|------------------|-----------|
| `%d` | int |
| `%f` | float |
| `%lf` | double |
| `%c` | char |
| `%s` | string (character array) |
| `%u` | unsigned int |

# 16. Common Beginner Mistakes

### Forgetting `&`

❌ Incorrect

```c
scanf("%d", age);
```

✔ Correct

```c
scanf("%d", &age);
```

---

### Wrong Format Specifier

```c
float salary;

scanf("%d", &salary);
```

This is incorrect because `%d` expects an integer.

---

### Missing Header File

Forgetting:

```c
#include <stdio.h>
```

may cause compiler warnings or errors.

---

### Using `%c` Without a Leading Space

```c
scanf("%c", &grade);
```

After previous input, `%c` may read the leftover newline character instead of the intended input.

Using:

```c
scanf(" %c", &grade);
```

avoids this common issue.

---

# 17. Best Practices

- Always use the correct format specifier.
- Use `&` with variables like `int`, `float`, `double`, and `char`.
- Display a clear prompt before calling `scanf()`.
- Validate user input in real-world applications.
- Keep input and output messages user-friendly.

# 19. Key Points to Remember

- `scanf()` reads input from the keyboard.
- It is declared in `stdio.h`.
- Use the correct format specifier for each data type.
- Most variables require the address-of operator (`&`) with `scanf()`.
- `%c` often benefits from a leading space to ignore leftover whitespace.

# 20. Summary

`scanf()` is one of the most important input functions in C. It allows users to provide values while the program is running, making programs interactive. By using the appropriate format specifiers and the address-of operator, `scanf()` stores user input in variables for later use in the program.
