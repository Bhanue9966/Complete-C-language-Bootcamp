# Assignment Operators

## 1. Introduction

Imagine you have an empty box.

```
+------------+
|            |
|            |
+------------+
```

Now, you place the number **25** inside it.

```
+------------+
|     25     |
+------------+
```

Similarly, in C programming, variables are like containers (boxes) that store data.

To place a value into a variable, C uses **Assignment Operators**.

Without assignment operators, variables would remain empty and could not store any data.

---

# 2. Definition

An **Assignment Operator** is a special symbol used to assign a value or the result of an expression to a variable.

The most commonly used assignment operator is:

```c
=
```

Example:

```c
int age;

age = 25;
```

The value `25` is assigned to the variable `age`.

---

# 3. Why Do We Need Assignment Operators?

Suppose you're developing a Student Management System.

The program should store:

- Student Name
- Student Age
- Marks
- Percentage
- Grade

To store these values in variables, assignment operators are required.

Without them, the program cannot remember or update data.

---

# 4. Types of Assignment Operators

| Operator | Name | Meaning | Example |
|----------|------|---------|---------|
| `=` | Assignment | Assign value | `a = 10` |
| `+=` | Add and Assign | `a = a + b` | `a += b` |
| `-=` | Subtract and Assign | `a = a - b` | `a -= b` |
| `*=` | Multiply and Assign | `a = a * b` | `a *= b` |
| `/=` | Divide and Assign | `a = a / b` | `a /= b` |
| `%=` | Modulus and Assign | `a = a % b` | `a %= b` |

# 6. Simple Assignment Operator (`=`)

The `=` operator assigns the value on the **right-hand side (RHS)** to the variable on the **left-hand side (LHS)**.

### Syntax

```c
variable = value;
```

Example

```c
salary = 50000;
```

The value `50000` is stored in `salary`.

---

# 7. Example 1 – Simple Assignment

```c
#include <stdio.h>

int main()
{
    int age;

    age = 22;

    printf("Age = %d", age);

    return 0;
}
```

### Output

```
Age = 22
```

### Explanation

The assignment operator stores the value `22` in the variable `age`.

---

### Conclusion

The `=` operator stores a value in a variable.

---

# 8. Compound Assignment Operators

Sometimes we want to update a variable based on its current value.

Instead of writing:

```c
marks = marks + 10;
```

we can simply write:

```c
marks += 10;
```

This makes the code shorter and easier to read.

---

# 9. Addition Assignment (`+=`)

Example

```c
#include <stdio.h>

int main()
{
    int salary = 30000;

    salary += 5000;

    printf("%d", salary);

    return 0;
}
```

### Output

```
35000
```

### Equivalent Statement

```c
salary = salary + 5000;
```

# 17. Common Beginner Mistakes

## Mistake 1

Confusing `=` with `==`

Incorrect

```c
if(age = 18)
```

Correct

```c
if(age == 18)
```

`=` assigns a value.

`==` compares two values.

---

## Mistake 2

Using an uninitialized variable.

Incorrect

```c
int x;

x += 10;
```

Since `x` has no initial value, the behavior is undefined.

Correct

```c
int x = 0;

x += 10;
```

---

## Mistake 3

Forgetting that assignment changes the variable.

Example

```c
int x = 5;

x += 3;
```

Now `x` becomes `8`.

---

# 18. Best Practices

- Initialize variables before using compound assignment operators.
- Use compound assignments to make code concise.
- Do not confuse `=` with `==`.
- Use meaningful variable names.
- Keep expressions simple and readable.

# 19. Interview Questions

An assignment operator stores a value or the result of an expression into a variable.

# 20. Key Points to Remember

- Assignment operators store values in variables.
- `=` is the basic assignment operator.
- Compound assignment operators combine an operation with assignment.
- `a += b` is equivalent to `a = a + b`.
- Initialize variables before using compound assignment operators.

# 21. Summary

Assignment operators are essential in C programming because they allow variables to store and update values. While the simple assignment operator (`=`) stores a value directly, compound assignment operators (`+=`, `-=`, `*=`, `/=`, `%=`) combine arithmetic operations with assignment, resulting in cleaner and more efficient code.
