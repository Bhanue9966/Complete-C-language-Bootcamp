# Arithmetic Operators

## 1. Introduction

Imagine you're using a calculator.

You perform operations like:

- Addition
- Subtraction
- Multiplication
- Division

Similarly, C provides **Arithmetic Operators** to perform mathematical calculations.

These operators are used in almost every C program, from simple calculators to complex banking and scientific applications.

---

# 2. Definition

Arithmetic operators are special symbols used to perform mathematical operations on numeric values (operands).

They are used to:

- Add numbers
- Subtract numbers
- Multiply numbers
- Divide numbers
- Find the remainder after division

---

# 3. Why Do We Need Arithmetic Operators?

Suppose you're developing a Student Management System.

The program should calculate:

- Total Marks
- Average Marks
- Percentage
- Remaining Fees
- Salary
- Tax

Without arithmetic operators, these calculations would not be possible.

### Important Note

### Important Note

# 17. Common Beginner Mistakes

## Mistake 1

Expecting

```c
5 / 2
```

to produce

```
2.5
```

Actual result

```
2
```

because both operands are integers.

---

## Mistake 2

Using modulus with floating-point numbers.

Incorrect

```c
5.5 % 2
```

Correct

```c
5 % 2
```

---

## Mistake 3

Division by Zero

```c
10 / 0
```

This results in **undefined behavior**. On most systems, the program will terminate with a runtime error.

Always ensure the divisor is not zero.

---

## Mistake 4

Ignoring Data Types

```c
float result = 5 / 2;
```

Many beginners expect

```
2.5
```

Actual result

```
2.0
```

because the division happens first using integers.

Correct

```c
float result = 5.0 / 2;
```

Output

```
2.5
```

---

# 18. Best Practices

- Choose the correct arithmetic operator.
- Check for division by zero before dividing.
- Use floating-point types when decimal precision is required.
- Use `%` only with integer operands.
- Write clear arithmetic expressions.
