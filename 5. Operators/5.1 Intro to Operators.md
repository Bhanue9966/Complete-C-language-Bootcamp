# Introduction to Operators

## 1. Introduction

Imagine you're using a calculator.

To perform calculations, you press symbols like:

```
+

-

×

÷
```

These symbols tell the calculator **what operation should be performed**.

Similarly, in C programming, we use **operators** to perform different operations on data.

Without operators, a program would only be able to store data—it could not calculate, compare, or make decisions.

---

# 2. What is an Operator?

An **operator** is a special symbol that performs an operation on one or more operands.

In simple words,

> An operator tells the compiler **what action to perform**.

Example:

```c
5 + 3
```

Here,

```
+

↓

Operator
```

It performs **addition**.

---

# 3. Definition

An **operator** is a symbol that instructs the compiler to perform a specific operation such as addition, subtraction, comparison, assignment, or logical evaluation on one or more operands.

---

# 4. Why Do We Need Operators?

Suppose you want to write a program that:

- Adds two numbers
- Finds the largest number
- Calculates percentage
- Checks voting eligibility
- Calculates salary
- Compares marks

Without operators, none of these tasks would be possible.

Operators make programs useful by enabling calculations and decision-making.

---

# 5. Real-Life Analogy

Imagine a washing machine.

```
Clothes

↓

Washing Machine

↓

Clean Clothes
```

The washing machine performs an **operation** on the clothes.

Similarly,

```
5

+

3

↓

8
```

The operator performs an operation on the values.

---

# 6. What are Operands?

The values on which an operator works are called **operands**.

Example:

```c
10 + 20
```

```
10

↓

Operand
```

```
+

↓

Operator
```

```
20

↓

Operand
```

So,

```
Operand

Operator

Operand
```

---

# 7. Operator vs Operand

Example:

```c
a + b
```

| Part | Meaning |
|------|----------|
| a | Operand |
| + | Operator |
| b | Operand |

---

# 8. What is an Expression?

An **expression** is a combination of operands, operators, constants, variables, and function calls that produces a value.

Example:

```c
a + b
```

```c
x * y
```

```c
marks / total
```

Each of these is an expression because it produces a result.

---

# 9. Expression Evaluation

Suppose,

```c
5 + 10
```

The compiler performs:

```
5

↓

Read First Operand

↓

+

↓

Perform Addition

↓

10

↓

Read Second Operand

↓

15
```

This process is called **expression evaluation**.

---

# 10. Types of Operators in C

C provides many categories of operators.

| Operator Type | Purpose |
|---------------|---------|
| Arithmetic | Mathematical calculations |
| Relational | Compare values |
| Logical | Combine conditions |
| Assignment | Assign values |
| Increment / Decrement | Increase or decrease values |
| Bitwise | Operate on binary bits |
| Conditional | Short form of if-else |
| sizeof | Find memory size |
| Comma | Evaluate multiple expressions |
| Pointer Operators | Work with memory addresses |

In this module, we'll focus on the most commonly used operator categories.

---

# 11. Example 1 – Addition

```c
#include <stdio.h>

int main()
{
    int a = 10;
    int b = 20;

    int sum = a + b;

    printf("%d", sum);

    return 0;
}
```

### Output

```
30
```

### Explanation

The `+` operator adds the values of `a` and `b`, and stores the result in `sum`.

---

### Conclusion

Operators enable mathematical calculations in C programs.

---

# 12. Example 2 – Comparison

```c
#include <stdio.h>

int main()
{
    int a = 10;
    int b = 20;

    printf("%d", a < b);

    return 0;
}
```

### Output

```
1
```

### Explanation

The `<` operator compares two values.

Since `10` is less than `20`, the result is `1` (true).

---

# 13. Example 3 – Assignment

```c
#include <stdio.h>

int main()
{
    int age;

    age = 25;

    printf("%d", age);

    return 0;
}
```

### Output

```
25
```

### Explanation

The `=` operator assigns the value `25` to the variable `age`.

---

# 14. Example 4 – Logical Operator

```c
#include <stdio.h>

int main()
{
    int age = 22;

    printf("%d", age >= 18 && age <= 60);

    return 0;
}
```

### Output

```
1
```

### Explanation

The `&&` operator combines two conditions.

Both are true, so the result is `1`.

---

# 16. Common Beginner Mistakes

## Mistake 1

Confusing operators with operands.

Example:

```c
10 + 20
```

```
10

Operand
```

```
+

Operator
```

```
20

Operand
```

---

## Mistake 2

Thinking operators work only with numbers.

Incorrect.

Operators can also work with:

- Characters
- Variables
- Pointers
- Expressions

---

## Mistake 3

Ignoring operator precedence.

Example:

```c
5 + 3 * 2
```

Many beginners expect:

```
16
```

Actual result:

```
11
```

because `*` has higher precedence than `+`.

We'll study this in detail later.

---

# 17. Best Practices

- Choose the correct operator for each task.
- Use parentheses to improve readability.
- Avoid writing overly complex expressions.
- Learn operator precedence to prevent unexpected results.
- Test expressions with sample values.
