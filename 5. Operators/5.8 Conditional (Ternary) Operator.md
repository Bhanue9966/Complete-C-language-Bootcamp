# Conditional (Ternary) Operator

## 1. Introduction

Suppose you want to determine the larger of two numbers.

Using an `if-else` statement:

```c
if (a > b)
{
    max = a;
}
else
{
    max = b;
}
```

C provides a shorter way to write this logic using the **Conditional (Ternary) Operator**.

```c
max = (a > b) ? a : b;
```

Both statements produce the same result, but the second one is more concise.

---

# 2. Definition

The **Conditional Operator (`?:`)** is the only operator in C that takes **three operands**.

It evaluates a condition and returns one of two expressions depending on whether the condition is true or false.

Because it uses three operands, it is also called the **Ternary Operator**.

---

# 3. Why Do We Need the Conditional Operator?

Imagine developing an online shopping application.

You need to determine:

- The higher discount
- The maximum salary
- Whether a student passed or failed
- Whether a person is eligible to vote

Instead of writing a full `if-else` block for simple decisions, the conditional operator provides a compact alternative.

---

# 4. Syntax

```c
condition ? expression_if_true : expression_if_false;
```

### Components

| Part | Description |
|------|-------------|
| `condition` | Expression evaluated first |
| `?` | Separates the condition from the true expression |
| `expression_if_true` | Executed if the condition is true |
| `:` | Separates the true and false expressions |
| `expression_if_false` | Executed if the condition is false |

---

# 5. How It Works

Example:

```c
marks >= 35 ? "Pass" : "Fail"
```

Execution Flow

```
Evaluate Condition

↓

marks >= 35

↓

Is it True?

↓

Yes ----------------→ Return "Pass"

↓

No

↓

Return "Fail"
```

---

# 6. Example 1 – Find the Larger Number

```c
#include <stdio.h>

int main()
{
    int a = 25;
    int b = 40;

    int max = (a > b) ? a : b;

    printf("Largest = %d", max);

    return 0;
}
```

### Output

```
Largest = 40
```

### Explanation

```
25 > 40

↓

False

↓

Return b

↓

40
```

---

### Conclusion

The conditional operator returns the larger value without using an `if-else` statement.

# 10. Nested Conditional Operator

The conditional operator can also be nested.

Example:

```c
#include <stdio.h>

int main()
{
    int a = 15;
    int b = 25;
    int c = 20;

    int largest = (a > b)
                    ? ((a > c) ? a : c)
                    : ((b > c) ? b : c);

    printf("Largest = %d", largest);

    return 0;
}
```

### Output

```
Largest = 25
```

> **Note:** Although nesting is possible, deeply nested conditional operators reduce readability. In such cases, an `if-else` statement is usually a better choice.

---

# 11. Conditional Operator vs if-else

| Conditional Operator | if-else |
|----------------------|----------|
| Short and concise | More readable for complex logic |
| Best for simple decisions | Best for multiple statements |
| Returns a value | Executes a block of statements |
| Single expression | Multiple statements allowed |

# 13. Common Beginner Mistakes

## Mistake 1

Forgetting the `:` symbol.

Incorrect

```c
a > b ? a;
```

Correct

```c
a > b ? a : b;
```

---

## Mistake 2

Using the conditional operator for complex logic.

Incorrect

```c
condition1
? statement1
: condition2
? statement2
: condition3
? statement3
: statement4;
```

This becomes difficult to read.

Prefer `if-else` for complex decision-making.

---

## Mistake 3

Using incompatible expression types.

Example

```c
int value = (condition) ? 10 : "Hello";
```

The two expressions should be type-compatible in typical use.

---

# 14. Best Practices

- Use the conditional operator for simple decisions.
- Prefer `if-else` when multiple statements are required.
- Avoid deeply nested conditional operators.
- Add parentheses around the condition to improve readability.

Example:

```c
result = (a > b) ? a : b;
```

# 15. Interview Questions

### 1. What is the conditional operator?

The conditional operator (`?:`) is a ternary operator that evaluates a condition and returns one of two expressions.

---

### 2. Why is it called a ternary operator?

Because it operates on **three operands**:

- Condition
- Expression if true
- Expression if false

---

### 3. What is the syntax of the conditional operator?

```c
condition ? expression1 : expression2;
```

---

### 4. Can the conditional operator replace `if-else`?

Yes, but only for **simple decisions** where a value is selected between two expressions.

---

### 5. What is the output?

```c
int a = 8;
int b = 12;

printf("%d", (a > b) ? a : b);
```

Output

```
12
```

---

# 16. Key Points to Remember

- The conditional operator is written as `?:`.
- It is the only ternary operator in C.
- It evaluates a condition and returns one of two expressions.
- It is ideal for simple decision-making.
- Avoid excessive nesting.

---

# 17. Summary

The Conditional (Ternary) Operator provides a concise alternative to simple `if-else` statements by evaluating a condition and selecting one of two expressions. It improves code brevity for straightforward decisions, but readability should always take priority when logic becomes more complex.
