# Keywords

## 1. Introduction

Every language has certain words that have a special meaning.

For example, in English:

- if
- else
- while

These words already have predefined meanings.

Similarly, the C programming language contains a set of reserved words called **keywords**.

Each keyword has a predefined meaning that is understood by the compiler.

Because these words already have a special purpose, they **cannot be used as variable names, function names, or other identifiers**.

---

# 2. Definition

A **keyword** is a reserved word in C that has a predefined meaning and purpose.

Keywords are recognized by the compiler and are used to define the syntax and structure of C programs.

---

# 3. Why Do We Need Keywords?

Imagine driving on a road.

Traffic signs already have fixed meanings.

```
STOP

↓

Must Stop
```

```
SPEED LIMIT

↓

Maximum Allowed Speed
```

You cannot decide that a STOP sign means "Go".

Similarly, in C:

```
int

↓

Integer Data Type
```

```
return

↓

Exit a Function
```

```
if

↓

Decision Making
```

Their meanings are fixed and cannot be changed.

---

# 4. Real-Life Analogy

Think about a classroom.

The teacher says:

```
Stand Up
```

Everyone knows what to do.

If every student interpreted "Stand Up" differently, the classroom would become chaotic.

Keywords work in the same way.

They provide standard instructions that every C compiler understands.

---

# 5. Characteristics of Keywords

Keywords have the following properties:

- Reserved by the C language.
- Have predefined meanings.
- Cannot be used as identifiers.
- Are recognized by the compiler.
- Are written in lowercase in standard C.

Example:

```c
int age;
```

Here:

```
int
```

is a keyword.

---

The following are the **32 keywords defined by the ANSI C (C90) standard**:

| Keyword | Purpose |
|----------|---------|
| `auto` | Automatic storage class |
| `break` | Exit a loop or switch |
| `case` | One branch of a switch statement |
| `char` | Character data type |
| `const` | Read-only object qualifier |
| `continue` | Skip to the next loop iteration |
| `default` | Default branch in a switch statement |
| `do` | Start of a do-while loop |
| `double` | Double-precision floating-point type |
| `else` | Alternative branch of an `if` statement |
| `enum` | Enumeration type |
| `extern` | External linkage/storage class |
| `float` | Floating-point data type |
| `for` | Loop statement |
| `goto` | Jump to a labeled statement |
| `if` | Conditional statement |
| `int` | Integer data type |
| `long` | Type modifier |
| `register` | Register storage class |
| `return` | Return from a function |
| `short` | Type modifier |
| `signed` | Signed type modifier |
| `sizeof` | Size operator |
| `static` | Static storage class |
| `struct` | Structure type |
| `switch` | Multi-way selection statement |
| `typedef` | Create a type alias |
| `union` | Union type |
| `unsigned` | Unsigned type modifier |
| `void` | No value or no type |
| `volatile` | Prevent certain compiler optimizations |
| `while` | Loop statement |

> **Note:** Modern C standards (such as C99, C11, C17, and C23) introduce additional keywords. Since you're learning C from the basics, we'll first master the classic ANSI C keywords and later discuss newer additions where relevant.

# 7. Examples of Keywords

## Example 1

```c
int age = 20;
```

Keywords:

```text
int
```

Explanation:

`int` tells the compiler that `age` is an integer variable.

---

## Example 2

```c
if (age >= 18)
{
    printf("Eligible");
}
```

Keyword:

```text
if
```

Explanation:

`if` allows the program to make a decision based on a condition.

---

## Example 3

```c
return 0;
```

Keyword:

```text
return
```

Explanation:

`return` ends the current function and optionally sends a value back to the caller.

---

# 8. Can Keywords Be Used as Variable Names?

No.

Example:

❌ Incorrect

```c
int int = 10;
```

Compiler Error:

```text
error: expected identifier before 'int'
```

Reason:

The compiler already recognizes `int` as a keyword.

---

Correct:

```c
int number = 10;
```

---

# 9. Keywords vs Identifiers

| Keywords | Identifiers |
|----------|-------------|
| Reserved words | User-defined names |
| Predefined meaning | Programmer chooses the name |
| Cannot be modified | Can be created by the programmer |
| Cannot be used as variable names | Used for variables, functions, arrays, etc. |

Example:

```c
int marks;
```

Here:

```
int
```

is a keyword.

```
marks
```

is an identifier.

---

# 10. Behind the Scenes

Consider:

```c
int age = 20;
```

Conceptually, the compiler interprets it as:

```
Keyword

↓

int

↓

Allocate memory suitable for an integer

↓

Identifier

↓

age

↓

Store Value

↓

20
```

Keywords tell the compiler **what kind of operation or declaration** is being performed.

---

# 11. Common Beginner Mistakes

### Using Keywords as Variable Names

❌ Incorrect

```c
int if = 10;
```

---

### Incorrect Capitalization

❌ Incorrect

```c
Int age;
```

Compiler Error:

```text
unknown type name 'Int'
```

Reason:

C is case-sensitive.

The keyword is:

```c
int
```

not:

```c
Int
```

---

### Assuming All Reserved-Looking Words Are Keywords

Words like:

```text
main
printf
scanf
```

are **not** C keywords.

- `main` is the conventional entry-point function name.
- `printf()` and `scanf()` are library functions declared in `<stdio.h>`.

---

# 12. Language Rules

### Rule 1

Keywords cannot be used as identifiers.

✔ Correct

```c
int age;
```

❌ Incorrect

```c
int int;
```

---

### Rule 2

Keywords are case-sensitive.

```c
int
```

is valid.

```c
INT
```

is **not** a keyword.

---

### Rule 3

The meaning of a keyword cannot be changed by the programmer.

---

# 13. Best Practices

- Never attempt to use keywords as variable or function names.
- Write keywords in lowercase.
- Choose descriptive identifiers that do not resemble keywords.
- Learn the common keywords (`int`, `if`, `for`, `while`, `return`, `const`) early—they appear in almost every C program.

# 15. Key Points to Remember

- Keywords are reserved words.
- They have predefined meanings.
- Keywords cannot be used as identifiers.
- C keywords are case-sensitive.
- `printf()` and `scanf()` are functions, not keywords.
- Modern C adds more keywords, but the original ANSI C set contains 32.

---

# 16. Summary

Keywords are predefined reserved words that form the foundation of the C programming language. They define data types, control structures, storage classes, and other language features. Since their meanings are fixed, they cannot be used as identifiers. Understanding keywords is essential because every C program relies on them to express its logic and structure.
