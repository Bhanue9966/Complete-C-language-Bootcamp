# int Data Type

## 1. Introduction

In our daily life, we frequently work with whole numbers.

Examples:

- Age = 21
- Number of students = 60
- Roll Number = 105
- Total books = 15
- Number of employees = 250

These values do not contain decimal points.

In C programming, such whole numbers are stored using the **`int`** data type.

The word **`int`** stands for **Integer**.

It is one of the most commonly used data types in C.

---

# 2. Definition

An **`int`** is a fundamental data type in C used to store **whole numbers (integers)** without any decimal or fractional part.

Examples:

```
10

250

-35

0

9999
```

All of the above are integer values.

---

# 3. Why Do We Need `int`?

Imagine a school management system.

It stores:

- Student Roll Number
- Student Age
- Total Students
- Number of Subjects

All these values are whole numbers.

Using a decimal data type would waste memory and be unnecessary.

The `int` data type efficiently stores whole numbers.

---

# 4. Real-Life Analogy

Imagine counting chairs in a classroom.

You count:

```
1

2

3

...

50
```

You never say:

```
25.75 chairs
```

because chairs are counted as whole units.

Similarly,

```
Age

Marks

Roll Number

Population Count

Quantity
```

are usually stored using `int`.

# 6. Memory Size

Typical size:

```
4 Bytes
```

which equals

```
32 Bits
```

> **Important:** The exact size of `int` is implementation-defined. On most modern systems it is 4 bytes, but it may differ depending on the compiler and hardware.

---

# 7. Value Range

On most systems where `int` is 4 bytes:

```
Minimum

-2,147,483,648
```

```
Maximum

2,147,483,647
```

If a value exceeds this range, the behavior depends on the operation and implementation. Signed integer overflow should be avoided because it results in **undefined behavior** in C.

---

# 8. Format Specifier

For `printf()`:

```c
%d
```

or

```c
%i
```

Both `%d` and `%i` print an integer value with `printf()`.

For `scanf()`:

```c
%d
```

Example:

```c
int age;

scanf("%d", &age);

printf("%d", age);
```

---

# 9. Example 1 – Store Age

```c
#include <stdio.h>

int main()
{
    int age = 21;

    printf("Age = %d", age);

    return 0;
}
```

### Output

```
Age = 21
```

### Explanation

The variable `age` stores the integer value `21`.

---

### Conclusion

Use `int` whenever the value is a whole number.

---

# 10. Example 2 – Addition

```c
#include <stdio.h>

int main()
{
    int a = 10;
    int b = 20;

    int sum = a + b;

    printf("Sum = %d", sum);

    return 0;
}
```

### Output

```
Sum = 30
```

### Explanation

The integer values are added together and the result is stored in another `int` variable.

---

### Conclusion

`int` is commonly used for arithmetic calculations.

---

# 11. Example 3 – User Input

```c
#include <stdio.h>

int main()
{
    int marks;

    printf("Enter your marks: ");

    scanf("%d", &marks);

    printf("Marks = %d", marks);

    return 0;
}
```

### Sample Output

```
Enter your marks: 95

Marks = 95
```

### Explanation

`scanf()` reads an integer from the keyboard and stores it in `marks`.

---

# 12. Example 4 – Integer Division

```c
#include <stdio.h>

int main()
{
    int a = 10;
    int b = 3;

    printf("%d", a / b);

    return 0;
}
```

### Output

```
3
```

### Explanation

Although

```
10 ÷ 3 = 3.333...
```

the result is

```
3
```

because both operands are integers.

The fractional part is discarded.

---

### Conclusion

Division between two integers produces an integer result.

---

# 13. Example 5 – Counting Loop

```c
#include <stdio.h>

int main()
{
    int i;

    for(i = 1; i <= 5; i++)
    {
        printf("%d\n", i);
    }

    return 0;
}
```

### Output

```
1
2
3
4
5
```

### Explanation

Loop counters are almost always declared as `int`.

# 16. Common Beginner Mistakes

## Mistake 1

Storing decimal values

```c
int price = 25.75;
```

Output

```
25
```

Reason:

The fractional part is discarded.

Correct:

```c
float price = 25.75f;
```

---

## Mistake 2

Using the wrong format specifier

```c
printf("%f", age);
```

Incorrect.

Correct:

```c
printf("%d", age);
```

---

## Mistake 3

Reading an integer without `&`

```c
scanf("%d", age);
```

Incorrect.

Correct:

```c
scanf("%d", &age);
```

---

## Mistake 4

Assuming all systems use 4-byte integers

Different systems may use different sizes for `int`.

Use `sizeof(int)` if your program depends on the size.

---

# 17. Best Practices

- Use `int` for whole numbers.
- Avoid storing decimal values in an integer.
- Use meaningful variable names.
- Use `%d` for input and output.
- Check for overflow when working with very large values.
- Use `sizeof(int)` if you need to know the size on the current platform.

# 18. Interview Questions

An `int` is a basic C data type used to store whole numbers.

---

### 2. What is the typical size of `int`?


or

```
%i
```

```
%d
```

# 20. Key Points to Remember

- `int` stores whole numbers.
- It cannot accurately store fractional values.
- The typical size is 4 bytes.
- `%d` is the most common format specifier.
- Integer division discards the fractional part.
- `int` is one of the most frequently used data types in C.

The `int` data type is the primary choice for storing whole numbers in C. It provides efficient storage for integer values and is widely used in calculations, counting, indexing, loops, and decision-making.
