# Data Types

## 1. Introduction

Imagine you are packing different items for shipping.

- Books are packed in a box.
- Water is stored in a bottle.
- Clothes are packed in a bag.
- Money is kept in a wallet.

Each item requires an appropriate type of container.

Similarly, in C programming, different kinds of data require different amounts of memory and different ways of being stored.

For example:

- An age is a whole number.
- A salary may contain decimal values.
- A grade is a single character.
- A person's name is a sequence of characters.

To store these different kinds of information correctly, C provides **data types**.

A data type tells the compiler **what kind of data is being stored**, **how much memory should be allocated**, and **what operations can be performed on it**.

---

# 2. Definition

A **data type** is a specification that tells the compiler what type of value a variable can store, how much memory should be allocated for it, and how the stored value should be interpreted.

---

# 3. Why Do We Need Data Types?

Suppose every variable occupied exactly the same amount of memory.

Consider these values:

```
'A'

25

3.1415926535

9876543212345
```

Each requires a different amount of storage.

If all variables used the same memory size:

- Small values would waste memory.
- Large values might not fit.

Data types solve this problem by allowing the compiler to allocate an appropriate amount of memory for each kind of data.

Benefits:

- Efficient memory usage.
- Better program performance.
- Correct interpretation of values.
- Type checking by the compiler.

# 6. Basic Data Types

The four primary data types in C are:

| Data Type | Used For |
|-----------|----------|
| `char` | Single character |
| `int` | Whole numbers |
| `float` | Decimal numbers (single precision) |
| `double` | Decimal numbers (double precision) |

Example:

```c
char grade = 'A';

int age = 21;

float salary = 45000.75f;

double pi = 3.141592653589793;
```

---

# 7. Type Modifiers

Type modifiers change the size or range of a basic data type.

The common modifiers are:

- `short`
- `long`
- `signed`
- `unsigned`

Examples:

```c
short int year;

long int population;

unsigned int count;

signed int temperature;
```

We'll study each modifier in detail later in this module.

---

# 8. Memory Size

Every data type occupies memory.

**Important:** The exact size depends on the compiler and the computer architecture.

Typical sizes on modern systems are:

| Data Type | Typical Size |
|-----------|--------------|
| `char` | 1 byte |
| `short` | 2 bytes |
| `int` | 4 bytes |
| `long` | 4 or 8 bytes |
| `float` | 4 bytes |
| `double` | 8 bytes |

> **Note:** The C standard specifies minimum requirements for data types, not exact sizes. Always use the `sizeof` operator if you need to know the size on a particular system.

---

# 9. Example Program

```c
#include <stdio.h>

int main()
{
    char grade = 'A';
    int age = 20;
    float marks = 95.5f;
    double salary = 45000.56789;

    printf("Grade  : %c\n", grade);
    printf("Age    : %d\n", age);
    printf("Marks  : %.1f\n", marks);
    printf("Salary : %.5lf\n", salary);

    return 0;
}
```

### Output

```
Grade  : A
Age    : 20
Marks  : 95.5
Salary : 45000.56789
```

### Explanation

- `char` stores a single character.
- `int` stores a whole number.
- `float` stores a decimal number with single precision.
- `double` stores a decimal number with higher precision.

### Conclusion

Different data types are used based on the nature of the data being stored.

# 12. Common Beginner Mistakes

### Using the Wrong Data Type

❌

```c
char age = 25;
```

Although this may compile, `char` is intended for character data and typically has a much smaller range than `int`.

✔ Better

```c
int age = 25;
```

---

### Storing Decimal Values in an Integer

```c
int price = 45.75;
```

The fractional part is discarded during conversion.

✔ Better

```c
float price = 45.75f;
```

---

### Using the Wrong Format Specifier

```c
float marks = 95.5;

printf("%d", marks);
```

Incorrect because `%d` is for integers.

✔ Correct

```c
printf("%f", marks);
```

---

# 13. Best Practices

- Choose the smallest appropriate data type that meets your requirements.
- Use `int` for whole numbers.
- Use `float` or `double` for decimal values.
- Use `char` for single characters.
- Use the correct format specifier with `printf()` and `scanf()`.
- Avoid unnecessary type conversions.

# 14. Interview Questions

### 1. What is a data type?

A data type specifies the kind of data a variable can store, the memory allocated for it, and the operations allowed on it.

---

### 2. Why are data types important?

They help the compiler allocate memory efficiently and correctly interpret stored values.

---

### 3. Name the four primary data types in C.

- `char`
- `int`
- `float`
- `double`

---

### 4. What are type modifiers?

Type modifiers (`short`, `long`, `signed`, `unsigned`) modify the size or range of basic data types.

---

### 5. Does C guarantee exact memory sizes for all data types?

No. The C standard specifies minimum requirements, while actual sizes depend on the compiler and architecture.

---

# 15. Key Points to Remember

- Every variable has a data type.
- Data types determine memory usage and value interpretation.
- C provides four primary data types: `char`, `int`, `float`, and `double`.
- Type modifiers adjust the size or range of numeric types.
- Memory sizes may vary between systems.

---

# 16. Summary

Data types are one of the core concepts of C programming. They define what kind of data a variable can store, how much memory it occupies, and how the compiler processes it. Choosing the correct data type leads to efficient memory usage, better performance, and more reliable programs.
