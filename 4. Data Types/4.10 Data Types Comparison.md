# Data Types Comparison

## 1. Introduction

After learning all the basic data types and type modifiers in C, it is important to compare them in one place.

This chapter acts as a **quick revision guide** that helps you understand:

- Memory size
- Value range
- Format specifiers
- Usage
- Advantages
- Limitations
- Differences between similar data types

> **Note:** The sizes and ranges shown below are **typical values**. The exact values depend on the compiler, operating system, and hardware architecture.

---

# 2. Comparison of Basic Data Types

| Feature | int | float | double | char |
|---------|-----|--------|---------|------|
| Category | Integer | Floating Point | Floating Point | Character |
| Typical Size | 4 Bytes | 4 Bytes | 8 Bytes | 1 Byte |
| Stores | Whole Numbers | Decimal Numbers | Large Decimal Numbers | Single Character |
| Decimal Values | ❌ No | ✅ Yes | ✅ Yes | ❌ No |
| Character Storage | ❌ No | ❌ No | ❌ No | ✅ Yes |
| Precision | Exact Integers | ~6–7 Decimal Digits | ~15–16 Decimal Digits | ASCII Character |
| Format Specifier | `%d` | `%f` | `%lf` | `%c` |
| Header File Required | `stdio.h` | `stdio.h` | `stdio.h` | `stdio.h` |
| Example | `100` | `95.75` | `123456.7890123` | `'A'` |
| Typical Use | Age, Marks | Temperature | Scientific Calculations | Letters |

# 6. Precision Comparison

| Type | Precision |
|------|-----------|
| float | Approximately 6–7 decimal digits |
| double | Approximately 15–16 decimal digits |

Example:

```c
float pi = 3.14159265358979;
double dpi = 3.14159265358979;
```

Output:

```
float
3.141593

double
3.141592653589790
```

# 10. Advantages and Limitations

| Data Type | Advantages | Limitations |
|------------|------------|-------------|
| char | Uses very little memory | Stores only one character |
| short | Saves memory | Small range |
| int | Fast and commonly used | Limited for very large numbers |
| long | Larger range | Size varies by platform |
| float | Less memory than double | Lower precision |
| double | High precision | Uses more memory |
| unsigned | Larger positive range | Cannot store negative values |

# 11. Selection Guide

| Requirement | Choose |
|-------------|--------|
| Store a Character | char |
| Store Small Integer | short |
| Store General Integer | int |
| Store Very Large Integer | long |
| Store Decimal Number | float |
| Store Highly Accurate Decimal | double |
| Store Only Positive Numbers | unsigned |
| Store Positive and Negative Numbers | signed (default int) |

# 14. Summary

Each C data type is designed for a specific purpose. Choosing the appropriate type improves memory efficiency, performance, and code readability. Integer types (`short`, `int`, `long`) are used for whole numbers, floating-point types (`float`, `double`) are used for decimal values, `char` stores characters, and modifiers like `signed` and `unsigned` control how integer values are represented.

This comparison chapter serves as a quick reference and revision guide for all the data types covered in Module 4.
