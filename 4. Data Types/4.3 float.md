# float Data Type

## 1. Introduction

In our daily lives, we often work with numbers that contain decimal points.

Examples:

- Temperature = 36.5°C
- Height = 175.8 cm
- Weight = 68.75 kg
- Price = 499.99
- Interest Rate = 7.5%

These values are called **floating-point numbers**.

In C, such values are stored using the **`float`** data type.

The `float` data type is designed to store **real numbers** (numbers with fractional or decimal parts).

---

# 2. Definition

A **`float`** is a basic data type in C used to store **single-precision floating-point numbers**, which are numbers containing a decimal point.

Examples:

```
10.5

25.75

-3.14

0.0

99.99
```

---

# 3. Why Do We Need `float`?

Suppose you're developing an online shopping application.

The product price is:

```
₹499.99
```

If you store it as:

```c
int price = 499.99;
```

The stored value becomes:

```
499
```

The decimal part is lost.

To preserve decimal values, we use the `float` data type.

---

# 4. Real-Life Analogy

Imagine measuring the height of a person.

Instead of saying:

```
175 cm
```

you measure more accurately:

```
175.8 cm
```

Similarly:

- Weight = 68.5 kg
- Temperature = 36.7°C
- Fuel = 12.75 liters

These measurements require decimal precision, making `float` the appropriate choice.

---

# 5. Syntax

## Declaration

```c
float variableName;
```

Example:

```c
float salary;
```

---

## Initialization

```c
float salary = 45000.50f;
```

> **Note:** The suffix `f` indicates that the literal is a `float`. Without `f`, a decimal literal is treated as a `double` by default.

---

## Multiple Variables

```c
float length, width, height;
```

---

## Multiple Initialization

```c
float x = 5.5f, y = 8.2f;
```

---

# 6. Memory Size

Typical size:

```
4 Bytes
```

Equivalent to:

```
32 Bits
```

> **Important:** The exact representation depends on the compiler and hardware, but on most modern systems, `float` occupies 4 bytes.

---

# 7. Approximate Value Range

Typical range:

```
±1.2 × 10^-38

to

±3.4 × 10^38
```

This allows `float` to store very small and very large decimal values.

---

# 8. Precision

A `float` provides approximately:

```
6 to 7 decimal digits
```

Example:

```c
float pi = 3.1415926535f;
```

Stored value (approximate):

```
3.141593
```

The remaining digits are rounded due to limited precision.

---

# 9. Format Specifier

For `printf()`:

```c
%f
```

For `scanf()`:

```c
%f
```

Example:

```c
float marks;

scanf("%f", &marks);

printf("%f", marks);
```

---

# 10. Example 1 – Store Temperature

```c
#include <stdio.h>

int main()
{
    float temperature = 36.5f;

    printf("Temperature = %.1f°C", temperature);

    return 0;
}
```

### Output

```
Temperature = 36.5°C
```

### Explanation

The variable stores a decimal value representing body temperature.

---

### Conclusion

Use `float` for measurements that include decimal values.

---

# 11. Example 2 – Product Price

```c
#include <stdio.h>

int main()
{
    float price = 499.99f;

    printf("Price = ₹%.2f", price);

    return 0;
}
```

### Output

```
Price = ₹499.99
```

### Explanation

The format specifier `%.2f` displays two digits after the decimal point.

---

# 12. Example 3 – User Input

```c
#include <stdio.h>

int main()
{
    float weight;

    printf("Enter your weight: ");

    scanf("%f", &weight);

    printf("Weight = %.2f kg", weight);

    return 0;
}
```

### Sample Output

```
Enter your weight: 68.75

Weight = 68.75 kg
```

# 13. Example 4 – Average of Marks

```c
#include <stdio.h>

int main()
{
    float m1 = 85.5f;
    float m2 = 92.5f;

    float average = (m1 + m2) / 2;

    printf("Average = %.2f", average);

    return 0;
}
```

### Output

```
Average = 89.00
```

### Explanation

The average includes a decimal value, making `float` suitable.

---

# 14. Example 5 – Area of a Circle

```c
#include <stdio.h>

int main()
{
    float radius = 5.0f;

    printf("Area = %.2f", area);

    return 0;
}
```

### Output

```
Area = 78.50
```

### Explanation

Mathematical formulas often produce decimal values, so `float` is commonly used.

# 17. Common Beginner Mistakes

## Mistake 1

Using `%d`

```c
float marks = 95.5;

printf("%d", marks);
```

Incorrect.

Correct:

```c
printf("%f", marks);
```

---

## Mistake 2

Forgetting the `&` in `scanf()`

```c
scanf("%f", weight);
```

Incorrect.

Correct:

```c
scanf("%f", &weight);
```

---

## Mistake 3

Comparing Floating-Point Numbers Directly

```c
float a = 0.1f + 0.2f;

if(a == 0.3f)
```

Direct comparison may not behave as expected because floating-point values are stored approximately.

---

## Mistake 4

Expecting Exact Precision

```c
float pi = 3.14159265358979f;
```

A `float` cannot accurately store all these digits.

---

# 18. Best Practices

- Use `float` for decimal values where single precision is sufficient.
- Add the `f` suffix to floating-point literals intended for `float`.
- Use appropriate format specifiers.
- Avoid direct equality comparisons for floating-point values.
- Use `double` if higher precision is required.

---

# 19. When to Use `float`

| Situation | Recommended |
|-----------|-------------|
| Temperature | `float` |
| Weight | `float` |
| Height | `float` |
| Percentage | `float` |
| Product Price | `float` |
| Scientific Measurements | `float` (or `double` if higher precision is needed) |

---

# 20. `int` vs `float`

| Feature | `int` | `float` |
|---------|------|---------|
| Stores | Whole numbers | Decimal numbers |
| Decimal Values | ❌ No | ✅ Yes |
| Typical Size | 4 Bytes | 4 Bytes |
| Format Specifier | `%d` | `%f` |
| Precision | Exact for integers within range | Approx. 6–7 decimal digits |

# 21. Interview Questions

### 1. What is a `float`?

A `float` is a basic data type used to store decimal (floating-point) numbers.

---

### 2. What is the typical size of `float`?

Typically **4 bytes**.

---

### 3. Which format specifier is used for `float`?

```
%f
```

---

### 4. Why do we write `5.5f` instead of `5.5`?

Because decimal literals are of type `double` by default. The `f` suffix specifies that the literal is a `float`.

---

### 5. Can `float` store all decimal values exactly?

No. `float` has limited precision, so some decimal values are stored approximately.

---

# 22. Did You Know?

💡 The value:

```c
0.1
```

cannot be represented exactly in binary floating-point.

This is why calculations involving decimal numbers may sometimes produce small rounding differences.

---

# 23. Key Points to Remember

- `float` stores decimal numbers.
- Typical size is 4 bytes.
- Precision is approximately 6–7 decimal digits.
- Use `%f` with `printf()` and `scanf()`.
- Add the `f` suffix to `float` literals.
- Avoid direct equality comparisons for floating-point values.

---

# 24. Summary

The `float` data type is used to store decimal values with single precision. It is widely used for measurements, prices, percentages, and scientific calculations where fractional values are required. Understanding its precision limits and correct usage is essential for writing reliable C programs.
