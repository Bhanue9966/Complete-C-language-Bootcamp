# char Data Type

## 1. Introduction

Not all information in a program is numeric.

Examples:

- Student Grade = A
- Gender = M
- Choice = Y
- Initial = B
- Symbol = $

These are **single characters**.

To store a single character, C provides the **`char`** data type.

The word **`char`** stands for **Character**.

Unlike `int`, `float`, and `double`, which store numbers, `char` stores a single character.

---

# 2. Definition

A **`char`** is a basic data type in C used to store a **single character**.

Examples:

```
'A'

'B'

'7'

'$'

'?'
```

Each value must be enclosed within **single quotation marks (`' '`)**.

---

# 3. Why Do We Need `char`?

Imagine a student management system.

Instead of storing:

```
Grade = 65
```

we often store:

```
Grade = 'A'
```

Similarly,

```
Gender = 'M'

Choice = 'Y'

Blood Group = 'O'
```

These values are single characters, making `char` the appropriate data type.

---

# 4. Real-Life Analogy

Imagine a keyboard.

Each key represents one character.

```
A

C

1

2

3

@

#

$
```

Pressing one key produces one character.

Likewise,

```c
char grade = 'A';
```

stores a single character.

# 6. Memory Size

Typical size:

```
1 Byte
```

Equivalent to:

```
8 Bits
```

Unlike other basic data types, a `char` occupies exactly **1 byte** in C.

---

# 7. Value Range

A `char` stores integer values internally.

The exact range depends on whether `char` behaves as signed or unsigned on your system.

Typical ranges are:

Signed `char`

```
-128 to 127
```

Unsigned `char`

```
0 to 255
```

> **Note:** Whether plain `char` is signed or unsigned is implementation-defined. We'll study `signed` and `unsigned` later in this module.

---

# 8. Format Specifier

For `printf()`:

```c
%c
```

For `scanf()`:

```c
%c
```

Example

```c
char grade;

scanf("%c", &grade);

printf("%c", grade);
```

---

# 9. Character Literals

Characters must be enclosed in **single quotes**.

Correct:

```c
'A'

'B'

'9'

'#'
```

Incorrect:

```c
"A"
```

Because:

```
'A'

↓

Character
```

```
"A"

↓

String
```

We'll learn about strings in a later module.

---

# 10. ASCII Representation

Computers do not actually store characters.

Instead, they store **ASCII codes** (American Standard Code for Information Interchange).

Examples:

| Character | ASCII Value |
|-----------|------------:|
| 'A' | 65 |
| 'B' | 66 |
| 'C' | 67 |
| 'a' | 97 |
| 'b' | 98 |
| '0' | 48 |
| '1' | 49 |
| ' ' (Space) | 32 |

Example:

```c
char ch = 'A';
```

Internally:

```
'A'

↓

65

↓

Stored in Memory
```

---

# 11. Example 1 – Store a Character

```c
#include <stdio.h>

int main()
{
    char grade = 'A';

    printf("Grade = %c", grade);

    return 0;
}
```

### Output

```
Grade = A
```

### Explanation

The variable `grade` stores a single character.

---

### Conclusion

Use `char` whenever you need to store one character.

---

# 12. Example 2 – Read a Character

```c
#include <stdio.h>

int main()
{
    char choice;

    printf("Enter Y or N: ");

    scanf("%c", &choice);

    printf("You entered: %c", choice);

    return 0;
}
```

### Sample Output

```
Enter Y or N: Y

You entered: Y
```

---

# 13. Example 3 – Print ASCII Value

```c
#include <stdio.h>

int main()
{
    char ch = 'A';

    printf("ASCII Value = %d", ch);

    return 0;
}
```

### Output

```
ASCII Value = 65
```

### Explanation

Although `ch` stores a character, it is internally represented by its ASCII value.

---

### Conclusion

Characters are stored as integers in memory.

---

# 14. Example 4 – ASCII to Character

```c
#include <stdio.h>

int main()
{
    char ch = 66;

    printf("%c", ch);

    return 0;
}
```

### Output

```
B
```

### Explanation

The integer value `66` corresponds to the ASCII character `'B'`.

---

# 15. Example 5 – Character Arithmetic

```c
#include <stdio.h>

int main()
{
    char ch = 'A';

    ch++;

    printf("%c", ch);

    return 0;
}
```

### Output

```
B
```

### Explanation

Internally:

```
'A'

↓

65

↓

65 + 1

↓

66

↓

'B'
```

---

### Conclusion

Characters can participate in arithmetic because they are stored as integers.

---

# 19. Common Beginner Mistakes

## Mistake 1

Using double quotes

```c
char ch = "A";
```

Incorrect.

Correct:

```c
char ch = 'A';
```

---

## Mistake 2

Using `%d`

```c
printf("%d", grade);
```

This prints the ASCII value instead of the character.

To print the character:

```c
printf("%c", grade);
```

---

## Mistake 3

Ignoring Whitespace with `scanf()`

After reading an integer, a newline character (`'\n'`) may remain in the input buffer.

Example:

```c
scanf("%c", &ch);
```

Sometimes reads the leftover newline instead of the intended character.

A common solution is:

```c
scanf(" %c", &ch);
```

Notice the space before `%c`, which tells `scanf()` to skip leading whitespace.

---

## Mistake 4

Trying to Store Multiple Characters

```c
char name = 'AB';
```

Incorrect.

A `char` stores only one character.

---

# 20. Best Practices

- Use single quotes for character literals.
- Use `%c` with `printf()` and `scanf()`.
- Remember that `char` stores ASCII values internally.
- Use `" %c"` in `scanf()` when appropriate to ignore leading whitespace.
- Use descriptive variable names like `grade`, `choice`, or `initial`.

# 22. `char` vs `int`

| Feature | `char` | `int` |
|---------|--------|-------|
| Stores | Single character | Whole numbers |
| Typical Size | 1 Byte | 4 Bytes |
| Format Specifier | `%c` | `%d` |
| Internal Storage | ASCII value | Integer value |

# 24. Did You Know?

💡 The following statement is perfectly valid:

```c
char ch = 'A';

printf("%d", ch);
```

Output:

```
65
```

This works because a `char` is automatically converted to its integer (ASCII) value when needed in an expression.

---

# 25. Key Points to Remember

- `char` stores one character.
- A `char` occupies 1 byte.
- Character literals use single quotes.
- `%c` is used for input and output.
- Characters are stored internally as integer values (such as ASCII).
- Character arithmetic is possible because of this internal representation.

---

# 26. Summary

The `char` data type is used to store single characters in C. Although it represents characters like `'A'` or `'9'`, it actually stores their corresponding integer codes internally. This unique property allows characters to participate in arithmetic operations and makes `char` the foundation for strings, text processing, and many advanced concepts in C programming.
